# **App Name**: Miaraka Health HR

## Core Features:

- Dashboard Overview: Provides a quick overview of the workforce's status, displaying key metrics such as total staff, currently present staff, and those on leave.
- AI-Driven Absence Prediction: Analyzes historical attendance data to predict the likelihood of employee lateness, highlighted visually to HR managers, with the LLM tool explaining possible reasons for potential lateness when hovering over an employee.
- Manual Correction Form: Allows HR to manually adjust employee time logs, requiring a reason for each correction to ensure accountability.
- Audit Trail: Maintains a detailed log of all manual corrections made, including who made the changes and when, displayed in an easily readable table.
- Biometric Log Import: Enables bulk uploading of biometric attendance data from .txt files, with immediate feedback on the success or failure of the import.
- Payroll Export: Generates payroll-ready reports with options to select the relevant period and preview summarized data before exporting to Excel.
- Shift Management: Enables HR staff to create, modify and assign the work schedule (Morning, Day, Night shift...) for the medical staff on a weekly basis using a planning-oriented display .

## Style Guidelines:

- Primary color: Slate blue (#64748B) to inspire trust and professionalism.
- Background color: Off-white (#F9FAFB) for a clean and modern interface.
- Accent color: Emerald green (#34D399) for positive status indicators and key actions.
- Font: 'Inter' sans-serif font for a modern and readable interface suitable for both headers and body text.
- Code font: 'Source Code Pro' for displaying log files in case of import errors.
- Use a consistent set of medical-themed icons (doctors, calendar, clock...) throughout the application, sourced from a set like FontAwesome.
- Fixed left sidebar navigation with collapsible sections for smaller screens.
